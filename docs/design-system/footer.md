# Component: Footer & pre-footer CTA

### Description

Footer oscuro + banda CTA pre-footer. Markup en `footer.php` (`.nvx-footer`, `.nvx-cta-banner`).

### Structure

- `.nvx-cta-banner` / `__inner` / `__title` / `__sub`  
- `.nvx-footer` / `__inner` / `__col` / `__bottom` / legal  

### Tokens

- shell + gutter-inner  
- labels: platinum  
- fondo: ink/charcoal dark  

### Home

Pre-footer CTA **oculto** en `body.home` (CSS home / banner rules).

### Ownership

**Solo** `nvx-footer.css`. Partials legacy `site-footer.php` **eliminados**.
